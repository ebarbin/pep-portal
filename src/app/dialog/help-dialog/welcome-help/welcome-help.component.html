Esta es la pantalla de bienvenida de la Plataforma educativa de Programación. En el margen superior de la pantalla podrás ver el menu principal. A continuacion describiremos las distintas opciones:

Cursos

Gestión de cursos. Podrás crear un curso nuevo, modificar existentes o eliminar alguno de ellos.

¿Que es un curso?
Representa una guía de ejercicios digital. Cada curso esta compuesto de:
  • Nombre
  • Descripción
  • Clave Solo los alumnos que sepan la clave podrán inscribirse al curso.
  • Ejercicios

Ejercicios

Gestión de ejercicios. Podrás crear un ejercicio nuevo, modificar existentes o eliminar alguno de ellos.

¿Que es un ejercicio?
Define una consigna, una practica, enunciado o un problema que debe ser resuelto por el alumno. Cada ejercicio esta compuesto de:
  • Nombre
  • Consigna
  • Primitivas Se pueden agregar primitivas para dejar disponibles al alumno.
  • Pre Ejecución Código javascript. Se pueden definir funciones, variables, datos, etc para dejar disponibles al alumno.
  • Pos Ejecución Código javascript. Se debe definir una función de validación. Si se valida automanticamente o si requiere corrección por parte del docente.

Primitivas

Gestión de primitivas. Podrás crear una primitiva nueva, modificar existentes o eliminar alguna de ellas.

¿Que es una primitiva?
Es una función javascript que debe ser sencilla y de utilidad para el alumno. Cada primitiva esta compuesta de:
  • Nombre
  • Descripción Debe describir el funcionamiento, sus parametros de entrada y su salida.
  • Función Código javascript.

Correcciones

Gestión de correciones. Podrás realizar correcciones sobre las soluciones de tus alumnos.


Consultas

Gestión de consultas. Podrás visualizar y responder las dudas de tus alumnos.


Estado de Resultados

Podrás visualizar el estado de tus cursos y el de tus alumnos.
